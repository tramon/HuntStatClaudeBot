# Hunt: Showdown 1896 — Knowledge Base
## Розділ: Weapons (Зброя)

> **Актуальність:** Update 2.8 "Road to Hell" (9 червня 2026)
>
> **⚠️ НОВА СИСТЕМА СЛОТІВ (Update 2.8):** Стара система Small/Medium/Large → числова **Size 1–5**.
> Ємність мисливця — **5 очок** (з Quartermaster — **6**). Два зброярні слоти.
>
> **🔴 Special Long Ammo:** Mosin, Berthier, Krag, Lebel — окремий тип набоїв. Між собою діляться, але НЕ зі Sparks / Martini / Uppercut. Поповнення лише з Special Ammo Crates, Ammo Boxes та Full Restocks.
>
> Варіанти (Marksman, Sniper, Deadeye, Silenced, Bayonet, Riposte, Talon, Trauma) мають той самий Size і той самий калібр, що й базова зброя — якщо не вказано окремо.

---

### 🔖 Легенда достовірності Size

| Значок | Значення |
|--------|----------|
| ✅ | Підтверджено офіційно (патчноути / офіційний блог Crytek) |
| 🔶 | Логічний висновок з офіційних правил ("carbines = 3, rifles = 4") |
| ❓ | Невідомо — повні патчноути ще не вийшли. **Перевір у грі!** |
